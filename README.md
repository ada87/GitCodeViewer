# GitCode Viewer

A high-performance source code reading tool, helping you view code anytime, anywhere.

You can search `GitCodeViewer` on GitHub in the **GitCode Viewer** app, or clone `https://github.com/ada87/GitCodeViewer.git` to try it out.

## Features

- **Offline Git:** View offline after cloning. Supports branch, log and other Git operations, and private repositories.
- **Syntax Highlight:** Supports mainstream languages (HTML, JS, CSS, Python, TypeScript, Java, C++, PHP, Rust, Go, Ruby, Kotlin, Dart, Bash, SQL, YAML, etc.).
- **Document Reading:** Supports Markdown, Mermaid (.mmd, .mermaid), Jupyter (.ipynb) and more.
- **Themes:** Supports Light/Dark modes and VS Code/JetBrains theme styles.
- **GitHub:** Quickly discover and download popular public projects.

# GitCode Viewer

Ein leistungsstarkes Quellcode-Lesewerkzeug, das Ihnen hilft, Code jederzeit und überall anzuzeigen.

## Hauptfunktionen

- **Offline-Git:** Nach dem Klonen offline anzeigen. Unterstützt branch, log und andere Git-Operationen.
- **Syntaxhervorhebung:** Unterstützt gängige Sprachen (HTML, JS, CSS, Python, TypeScript, Java, C++, PHP, Rust, Go, Ruby, Kotlin, Dart, Bash, SQL, YAML usw.).
- **Dokumentenlesen:** Unterstützt Markdown, Mermaid (.mmd, .mermaid), Jupyter (.ipynb) und mehr.
- **Themes:** Unterstützt Hell-/Dunkelmodus und VS Code-/JetBrains-Theme-Stile.
- **GitHub:** Entdecken und laden Sie schnell beliebte öffentliche Projekte herunter.

## Hinweis

Diese App ist nur ein Code-Reader; sie kann Code nicht bearbeiten oder ausführen.

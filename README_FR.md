# Visualiseur de Code Git

[English](./README.md) | [简体中文](./README_ZH.md) | [Deutsch](./README_DE.md) | [Español](./README_ES.md) | Français | [हिन्दी](./README_HI.md) | [日本語](./README_JA.md) | [한국어](./README_KO.md) | [Português (Brasil)](./README_PT_BR.md)

Un outil de lecture de code source haute performance qui vous aide à visualiser le code à tout moment et en tout lieu.

Dans l’app **GitCode Viewer**, vous pouvez rechercher `GitCodeViewer` sur GitHub ou cloner `https://github.com/ada87/GitCodeViewer.git` pour essayer.

## Fonctionnalités principales

- **Git hors ligne :** Visualisez hors ligne après le clonage. Prend en charge branch, log et autres opérations Git, ainsi que les dépôts privés.
- **Coloration syntaxique :** Prend en charge les principaux langages (HTML, JS, CSS, Python, TypeScript, Java, C++, PHP, Rust, Go, Ruby, Kotlin, Dart, Bash, SQL, YAML, etc.).
- **Lecture de documents :** Prend en charge Markdown, Mermaid (.mmd, .mermaid), Jupyter (.ipynb) et plus encore.
- **Thèmes :** Prend en charge les modes Clair/Sombre et les styles de thème VS Code/JetBrains.
- **GitHub :** Découvrez et téléchargez rapidement des projets publics populaires.

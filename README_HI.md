# Git कोड व्यूअर

एक उच्च-प्रदर्शन सोर्स कोड रीडिंग टूल है जो आपको कभी भी, कहीं भी कोड देखने में मदद करता है।

**GitCode Viewer** ऐप में, आप GitHub पर `GitCodeViewer` खोज सकते हैं या `https://github.com/ada87/GitCodeViewer.git` को क्लोन करके इसे आज़मा सकते हैं।

## विशेषताएं

- **ऑफ़लाइन Git:** क्लोन करने के बाद ऑफ़लाइन देखें। branch, log और अन्य Git संचालन के साथ निजी रिपॉज़िटरी का समर्थन करता है।
- **सिंटैक्स हाइलाइट:** मुख्य भाषाओं का समर्थन करता है (HTML, JS, CSS, Python, TypeScript, Java, C++, PHP, Rust, Go, Ruby, Kotlin, Dart, Bash, SQL, YAML, आदि)।
- **दस्तावेज़ पढ़ना:** Markdown, Mermaid (.mmd, .mermaid), Jupyter (.ipynb) और अधिक का समर्थन करता है।
- **थीम:** Light/Dark मोड और VS Code/JetBrains थीम स्टाइल का समर्थन करता है।
- **GitHub:** लोकप्रिय सार्वजनिक परियोजनाओं को तेज़ी से खोजें और डाउनलोड करें।

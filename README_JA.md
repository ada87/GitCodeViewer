# Gitコードビューアー

[English](./README.md) | [简体中文](./README_ZH.md) | [Deutsch](./README_DE.md) | [Español](./README_ES.md) | [Français](./README_FR.md) | [हिन्दी](./README_HI.md) | 日本語 | [한국어](./README_KO.md) | [Português (Brasil)](./README_PT_BR.md)

いつでもどこでもコードを表示できる高性能なソースコード閲覧ツールです。

**GitCode Viewer** アプリでは、GitHub で `GitCodeViewer` を検索するか、`https://github.com/ada87/GitCodeViewer.git` をクローンして試せます。

## 主な機能

- **オフライン Git:** クローン後はオフラインで表示。branch、log などの Git 操作とプライベートリポジトリをサポート。
- **シンタックスハイライト:** 主要な言語をサポート（HTML、JS、CSS、Python、TypeScript、Java、C++、PHP、Rust、Go、Ruby、Kotlin、Dart、Bash、SQL、YAML など）。
- **ドキュメント閲覧:** Markdown、Mermaid（.mmd、.mermaid）、Jupyter（.ipynb）などをサポート。
- **テーマ:** ライト/ダークモード、VS Code/JetBrains テーマスタイルをサポート。
- **GitHub:** 人気のある公開プロジェクトをすばやく発見してダウンロード。

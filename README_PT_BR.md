# Visualizador de Código Git

Uma ferramenta de leitura de código-fonte de alto desempenho, ajudando você a visualizar código a qualquer hora e em qualquer lugar.

No app **GitCode Viewer**, você pode pesquisar `GitCodeViewer` no GitHub ou clonar `https://github.com/ada87/GitCodeViewer.git` para testar.

## Recursos

- **Git Offline:** Visualize offline após clonar. Suporta branch, log e outras operações do Git, além de repositórios privados.
- **Destaque de Sintaxe:** Suporta linguagens principais (HTML, JS, CSS, Python, TypeScript, Java, C++, PHP, Rust, Go, Ruby, Kotlin, Dart, Bash, SQL, YAML, etc.).
- **Leitura de Documentos:** Suporta Markdown, Mermaid (.mmd, .mermaid), Jupyter (.ipynb) e muito mais.
- **Temas:** Suporta modos Claro/Escuro e estilos VS Code/JetBrains.
- **GitHub:** Descubra e baixe rapidamente projetos públicos populares.

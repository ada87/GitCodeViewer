# Git代码阅读器

一款高性能源码阅读工具，助您随时随地查看代码。

## 功能特性

- **离线Git：** 克隆之后离线查看，支持 branch、log 等 Git 操作，支持私有仓库。
- **语言高亮：** 支持几乎所有主流编程语言（HTML、JS、CSS、Python、TypeScript、Java、C++、PHP、Rust、Go、Ruby、Kotlin、Dart、Bash、SQL、YAML 等）。
- **文档阅读：** 支持 Markdown、Mermaid（.mmd、.mermaid）、Jupyter（.ipynb）等文档。
- **皮肤样式：** 支持明/暗皮肤，支持 VS Code/JetBrains 皮肤样式。
- **GitHub：** 快速发现并下载热门公开项目。

## 说明

本应用仅是一个代码阅读器，无法编辑、运行代码。

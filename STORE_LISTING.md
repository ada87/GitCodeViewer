# Google Play Store Listing

## 1. English (en)

**App Title:**
GitCode Viewer

**Short Description:**
Offline Git repo reader. Supports main languages, Markdown, diagrams, etc.

**Full Description:**
GitCode Viewer is a high-performance source code reading tool, helping you view code anytime, anywhere. Core features:

*   **Offline Git:** View offline after cloning. Supports branch, log, and other Git operations.
*   **Syntax Highlight:** Supports mainstream languages (HTML, JS, CSS, Python, TypeScript, Java, C++, PHP, Rust, Go, Ruby, Kotlin, Dart, Bash, SQL, YAML, etc.).
*   **Document Reading:** Supports Markdown, Mermaid (.mmd, .mermaid), Jupyter (.ipynb), and more.
*   **Themes:** Supports Light/Dark modes and VS Code/JetBrains theme styles.
*   **GitHub:** Quickly discover and download popular public projects.

**Note:** This app is a code reader only; it cannot edit or run code.

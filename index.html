About Code Viewer

Code Viewer is a mobile Git client designed exclusively for reading code. Clone repositories from GitHub, GitLab, or Bitbucket to your phone, then browse source code, read commits, switch branches, and explore file trees — all completely offline.

Offline-First Architecture

Once a repository is cloned, every file, every branch, and every commit is available without an internet connection. Read code on the subway, on a plane, or anywhere without signal. Pull updates when you reconnect.

Rich Code Rendering

Syntax highlighting for 24+ programming languages including TypeScript, Python, Go, Rust, Java, Kotlin, Swift, C++, and more. Choose between VS Code and JetBrains editor themes with both light and dark modes. Markdown files are rendered with full LaTeX math formula support. Mermaid diagrams (flowcharts, sequence diagrams, class diagrams) are visualized inline. Jupyter Notebooks (.ipynb) are rendered with code cells and outputs.

GitHub Discovery

Search GitHub repositories by keyword, language, stars, and topics directly from the app. Browse trending open-source projects and clone them with one tap.

Private Repository Access

Access private repositories from GitHub, GitLab, and Bitbucket using Personal Access Tokens. Credentials are encrypted and stored locally on your device. There are no backend servers — all network traffic goes directly between your phone and your Git provider.

Developer-Friendly Workspace

Open multiple files in tabs, navigate the file tree, and jump to any file instantly with the command palette. Browse commit history, switch branches, and view repository metadata. Available in 9 languages: English, Chinese, Japanese, Korean, German, Spanish, French, Hindi, and Portuguese.

Privacy & Security

Code Viewer collects zero user data. No analytics, no tracking, no telemetry. Your code and credentials never leave your device. Read the full privacy policy.
